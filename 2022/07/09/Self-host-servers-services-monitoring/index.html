Self-host servers & services - monitoring

Posted on 2022-07-09 Edited on 2025-12-04

Overview

Why we need to monitor our server and services?

In my personal experience, How can I quickly identify the root issue if I can’t get our password from bitwarden? Checking network connection, DNS resolution, and docker container status. It will always take lots of time to review each step; then A central monitoring service is indispensable for tracking the health of our server and services.

The main goal of monitoring any server is to save the server from any possible failure. The data collected by server monitoring can be beneficial in identifying and investigating problems. Server monitoring is the process of gaining visibility into the activity on your servers — whether physical or virtual.
